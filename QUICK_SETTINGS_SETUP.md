# 🚀 إنشاء جدول Settings - الطريقة السريعة

## ⏱️ الوقت المتوقع: 2 دقيقة

---

## الخطوات:

### 1️⃣ افتح Supabase
اذهب إلى [Supabase Dashboard](https://app.supabase.com)

---

### 2️⃣ اختر مشروعك
من القائمة الرئيسية، اختر مشروعك

---

### 3️⃣ اذهب إلى SQL Editor
من القائمة الجانبية → **SQL Editor**

---

### 4️⃣ اضغط New Query
اضغط على زر **New Query** أو **+**

---

### 5️⃣ انسخ الكود

افتح الملف `SUPABASE_SETTINGS_TABLE.sql` من مجلد المشروع

**انسخ كل الكود** (Ctrl+A ثم Ctrl+C)

---

### 6️⃣ الصق في SQL Editor

الصق الكود في محرر Supabase (Ctrl+V)

---

### 7️⃣ شغل الكود

اضغط **Run** أو اضغط **Ctrl+Enter**

---

### 8️⃣ تحقق من النتائج

يجب أن تظهر نتيجتان:

✅ **الأولى:** عدد الإعدادات = 1
```
settings_count
1
```

✅ **الثانية:** بيانات الإعدادات
```
restaurant_name | phone | email | address
```

---

## ✅ انتهى!

الجدول جاهز والبيانات موجودة. الآن تقدر:

1. اذهب إلى `/admin/settings`
2. عدّل البيانات
3. اضغط "حفظ الإعدادات"
4. البيانات ستُحفظ في Supabase تلقائياً ✨

---

## 🆘 إذا واجهت مشكلة

### ❌ الخطأ: "table already exists"
حذف الجدول القديم:
```sql
DROP TABLE IF EXISTS public.settings CASCADE;
```
ثم شغل Script كاملة من جديد

### ❌ لا تظهر البيانات
شيك:
```sql
SELECT * FROM public.settings;
```

### ❌ لا تعرف أيه الخطأ
انسخ رسالة الخطأ بالكامل واسأل
